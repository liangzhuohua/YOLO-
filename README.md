# YOLO系列数据集操作脚本
YOLO系列数据集操作脚本，使得制作数据集更加方便快速

## bing.py
bing浏览器爬虫 

## Compare_images_and_Labels.py 
解决图片与标签不匹配的问题 

## expand_pict.py 
数据集增强 

## getdata.py
爬取数据集图片 

## rename.py 
更改文件的名字 

## xmlTotxt.py 
XML格式标签转YOLO格式标签 

## yoloToVoc.py 
YOLO格式标签转VOC格式标签 

## 视频切帧.py 
把视频切成图片 

## 数据集划分.py 
把数据集划分为val、test、train三个部分 

## 数据集扩充.py 
数据集增强 

## 数据集扩充xml.py
数据集增强（xml格式数据集） 

## 修改txt数据集序号.py 
修改YOLO格式标签中的类别序号
